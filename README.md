# Transaction-Risk-Monitoring-And-KPI-Dashboard

## Overview

This project focuses on analyzing financial transaction data to identify potential fraud patterns, monitor transaction risks, and track key business performance indicators (KPIs). Using Excel, SQL, Python, and Tableau, the project provides actionable insights through interactive dashboards and data-driven reporting.

---

## Business Problem

Financial institutions and businesses process thousands of transactions daily, making it challenging to identify suspicious activities and monitor operational performance efficiently.

The objectives of this project were to:

* Detect high-risk and potentially fraudulent transactions.
* Monitor transaction trends and risk indicators.
* Track key performance metrics through dashboards.
* Support data-driven decision-making and risk management.
* Improve visibility into transaction behavior and business performance.

## Key KPIs

* Total Transactions
* Total Transaction Value
* Fraudulent Transaction Count
* Fraud Rate (%)
* High-Risk Transaction Volume
* Average Transaction Amount
* Monthly Transaction Trends
* Risk Category Distribution

---

## Key Insights

* Identified transaction patterns associated with elevated fraud risk.
* Monitored trends in transaction volumes and values across different periods.
* Tracked fraud-related KPIs to support risk management efforts.
* Developed interactive dashboards for performance monitoring and reporting.
* Generated actionable recommendations to improve transaction monitoring processes.
